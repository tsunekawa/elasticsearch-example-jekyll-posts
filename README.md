elasticsearch-example-jekyll-posts
==================================

Elasticsearch の練習用スクリプトです。
「[状況に埋め込まれたブログ](http://www.kunimiya.info/)」の記事データを使っています。

## Usage
0. まずElasticsearchを起動
1. ./bin/setup を実行
2. ./bin/post を実行
3. ./bin/search (検索キーワード)で検索
